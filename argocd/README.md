# Argo CD Bootstrap

This folder contains the Argo CD application structure for the platform repo.

Terraform in the `aws_infra` repo installs Argo CD and creates the root Application. The root Application points to:

```text
argocd/apps
```

The child Applications in this folder sync:

- Crossplane install
- Crossplane providers and functions
- Platform API XRD and Composition
- PlatformStack examples

Before use, replace `https://github.com/YOUR_ORG/YOUR_PLATFORM_REPO.git` with the real repository URL in the child Applications. The root Application generated by Terraform uses `platform_repo_url` from `variables/<env>/bootstrap.tfvars`.

The platform Argo CD values enable annotation-based tracking, which is recommended when Argo CD manages Crossplane resources.
